## Как импортировать

1. В n8n: **Settings → Import Workflow**
2. Загрузить `workflow-14-upgraded.json`
3. Загрузить `workflow-blog-upgraded.json`
4. Проверить credentials — они сохранены по ID, должны подцепиться
5. **Перед запуском** — выполнить все пункты из `MANUAL-TODO.md`
6. Тестовый запуск на одной теме
