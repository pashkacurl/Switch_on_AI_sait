## Быстрый старт

### Цветовая палитра
```
Фон:      #0a0c1c (тёмно-синий)
Текст:    #ffffff (белый)
Акцент 1: #5568f5 (синий)
Акцент 2: #ffd669 (жёлтый)
Акцент 3: #18E896 (зелёный)
```

### Шрифты
- Заголовки: **Montserrat** (700)
- Текст: **Inter** (400-500)

### Технологии
- WordPress + Astra Theme
- Elementor Pro
- GSAP + ScrollTrigger
- Spline (3D)

---

## Услуги на сайте

1. **Контент-завод** — генерация контента для соцсетей, статей, видео
2. **Чат-боты / автоворонки** — разработка и внедрение ботов
3. **Комплексная автоматизация** — экосистема AI-решений
4. **Обучение по AI** — курсы и тренинги по работе с ИИ
