## Ключевые метрики (KPI)

### Финансовые
- MRR (ежемесячная выручка)
- Средний чек
- LTV клиента
- CAC (стоимость привлечения)

### Продуктовые
- Количество студентов
- Завершаемость курсов
- NPS (удовлетворённость)
- Повторные покупки

### Маркетинговые
- Охваты в соцсетях
- Конверсия в лиды
- Конверсия в продажи
- Органический трафик
