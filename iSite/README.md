## О проекте

## Логическая связь документов

### Уровень 1: Фундамент (Стратегия)
**Стратегическая сессия** определяет личность основателя, ценности и 10-летнее видение.
- Файл: `01_Стратегия/Распаковка - брифинг 360...pdf`

### Уровень 2: Концепция (Бизнес)
**Бизнес-модель** формирует структуру экосистемы на основе видения.
- Файл: `02_Бизнес/Про мой бизнес.pdf`

### Уровень 3: Идентичность (Брендинг)
**Брендбук** визуализирует айдентику на основе архетипов личности.
- Файл: `03_Брендинг/Брендбук Включи ИИ.pdf`

### Уровень 4: Реализация (Операционные цели)
**Чек-листы** детализируют конкретные задачи для воплощения стратегии.
- Файл: `04_Операционные_цели/Чек-лист компании ВКЛЮЧИ ИИ.txt`

## Стандарты бренда

- **Простота** — сложные вещи объясняем просто
- **Результат с первого дня** — практика важнее теории
- **Честность** — обещаем только то, что можем дать
- **1 день = 1 инструмент** — измеримый прогресс
