# Switch On AI - WordPress Theme

Современная темная тема для WordPress с живыми 3D-анимациями в стиле Avtograf Group. Специально разработана для экосистемы AI-решений.

## 🎨 Особенности дизайна

- **Темная тема** с черным фоном (#000000)
- **Голубой акцент** (#69ace4) и кремовые оттенки
- **3D-анимации**: вращающиеся кольца, плавающий орб, анимированные точки
- **Blur-декорации** для глубины и атмосферы
- **Стекломорфизм** (glassmorphism) в элементах UI
- **Современная типографика**: Space Grotesk + Inter
- **Scroll-reveal анимации** при прокрутке страницы
- **Полностью адаптивный** дизайн для всех устройств

## 📋 Требования

- WordPress 6.0 или выше
- PHP 7.4 или выше
- Современный браузер с поддержкой CSS Grid и CSS Variables

## 🚀 Установка темы

### Вариант 1: Через админ-панель WordPress

1. Войдите в админ-панель WordPress
2. Перейдите в **Внешний вид → Темы**
3. Нажмите **Добавить новую** → **Загрузить тему**
4. Выберите ZIP-архив темы и нажмите **Установить**
5. После установки нажмите **Активировать**

### Вариант 2: Через FTP/хостинг

1. Загрузите папку `wordpress-theme` на ваш сервер
2. Разместите её в `/wp-content/themes/`
3. Переименуйте папку в `switchonai` (если нужно)
4. Перейдите в админ-панель WordPress → **Внешний вид → Темы**
5. Активируйте тему **Switch On AI**

### Вариант 3: Для WordPress.com

⚠️ **ВАЖНО**: На бесплатном плане WordPress.com нельзя устанавливать кастомные темы!

Для использования этой темы на WordPress.com необходим план **Business** или **eCommerce**.

Если у вас бесплатный/Personal план, рассмотрите альтернативы:
- Перенос на **WordPress.org** (самостоятельный хостинг)
- Использование кастомного CSS через **Внешний вид → Настроить → Дополнительные стили**
- Обновление до платного плана WordPress.com

## 🎨 Кастомизация стилей

### CSS переменные

Все цвета и настройки находятся в начале файла [style.css](wordpress-theme/style.css):

```css
:root {
    /* Backgrounds */
    --bg-primary: #000000;
    --bg-secondary: #0a0a0a;

    /* Accents */
    --accent-blue: #69ace4;
    --accent-cream: #f4dcc0;

    /* Typography */
    --font-heading: 'Space Grotesk', sans-serif;
    --font-body: 'Inter', sans-serif;
}
```

Измените эти переменные, чтобы настроить цветовую схему под ваш бренд.

### Добавление кастомного CSS

1. Перейдите в **Внешний вид → Настроить → Дополнительные стили**
2. Добавьте свой CSS код

## 🔧 Редактирование контента секций

Контент секций находится в файлах `template-parts/section-*.php`.

Чтобы изменить текст или структуру секции:

1. Откройте нужный файл (например, `section-services.php`)
2. Отредактируйте HTML-разметку и текст
3. Сохраните файл
4. Обновите страницу сайта

### Пример: Изменение услуг

Откройте `template-parts/section-services.php` и отредактируйте:

```php
<div class="service-card reveal">
    <div class="service-icon">📱</div>
    <h3>Ваша новая услуга</h3>
    <p>Описание услуги</p>
    <ul class="service-features">
        <li>Преимущество 1</li>
        <li>Преимущество 2</li>
        <li>Преимущество 3</li>
    </ul>
</div>
```

## 🐛 Решение проблем

### Анимации не работают

Убедитесь, что JavaScript файл подключен:
- Проверьте консоль браузера (F12)
- Очистите кеш сайта
- Проверьте, что файл `assets/js/main.js` существует

### Меню не отображается

1. Создайте меню в **Внешний вид → Меню**
2. Назначьте его в локацию "Primary Menu"
3. Добавьте пункты меню

### Стили не применяются

1. Очистите кеш браузера (Ctrl+Shift+Delete)
2. Очистите кеш WordPress (если используете плагин кеширования)
3. Проверьте права доступа к файлу `style.css`

### Футер пустой

Настройте виджеты в **Внешний вид → Виджеты** для трёх колонок футера.

## 📝 Дополнительные страницы

Для создания дополнительных страниц (О нас, Контакты и т.д.):

1. Создайте новую страницу в WordPress
2. Добавьте контент через редактор Gutenberg
3. Опубликуйте страницу

Страницы автоматически используют шаблон `index.php`.

## 🔐 Безопасность

Тема следует стандартам безопасности WordPress:

- Все данные экранируются через `esc_html()`, `esc_url()`, `esc_attr()`
- Используется `wp_enqueue_style()` и `wp_enqueue_script()`
