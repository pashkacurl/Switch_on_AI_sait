### WordPress.com vs WordPress.org

## Для WordPress.com

### Если у вас БЕСПЛАТНЫЙ/PERSONAL план

К сожалению, на бесплатном плане нельзя установить кастомную тему. У вас есть 3 варианта:

#### Вариант 1: Использовать кастомный CSS (частичное решение)

1. Войдите в админ-панель WordPress.com
2. Перейдите в **Внешний вид → Настроить → Дополнительные стили CSS**
3. Скопируйте весь CSS из файла [wordpress-theme/style.css](wordpress-theme/style.css)
4. Вставьте в поле "Дополнительные стили CSS"
5. Нажмите "Опубликовать"

⚠️ **Ограничения:**
- Вы сможете изменить только стили существующей темы
- HTML структура останется от выбранной темы WordPress.com
- Анимации могут не работать полностью

#### Вариант 2: Обновить до Business плана ($25/мес)

1. Перейдите в **Обновления → Планы**
2. Выберите план **Business** или **eCommerce**
3. После оплаты следуйте инструкциям из раздела [Business план](#если-у-вас-business-план)

#### Вариант 3: Перенести на WordPress.org (рекомендуется)

Это даст вам полный контроль и обойдётся дешевле. См. раздел [Для WordPress.org](#для-wordpressorg).

### Если у вас BUSINESS+ план

1. Войдите в админ-панель WordPress.com
2. Перейдите в **Мои сайты → Внешний вид → Темы**
3. Нажмите **Загрузить тему**
4. Создайте ZIP-архив папки `wordpress-theme`:
   ```bash
   # На Windows: щелкните правой кнопкой → Отправить → Сжатая ZIP-папка
   # Или используйте любой архиватор
   ```
5. Загрузите ZIP-файл
6. Активируйте тему
7. Настройте через **Внешний вид → Настроить**

## Для WordPress.org

### Шаг 1: Выбор хостинга

Рекомендуемые хостинги для России:

1. **Beget.com** - от 240₽/мес
   - Автоустановка WordPress
   - Бесплатный SSL
   - Поддержка на русском

2. **Timeweb.com** - от 199₽/мес
   - WordPress в 1 клик
   - Бесплатный домен первый год

3. **REG.RU** - от 290₽/мес
   - Готовые сборки WordPress
   - Круглосуточная поддержка

### Шаг 2: Установка WordPress

#### Через автоустановщик хостинга (рекомендуется)

1. Войдите в панель управления хостингом
2. Найдите раздел "Автоустановка CMS" или "WordPress"
3. Выберите "Установить WordPress"
4. Укажите домен и настройки
5. Дождитесь завершения установки
6. Запомните URL админ-панели, логин и пароль

#### Ручная установка

1. Скачайте WordPress с [ru.wordpress.org](https://ru.wordpress.org/download/)
2. Создайте базу данных MySQL на хостинге
3. Загрузите файлы WordPress через FTP
4. Откройте ваш домен в браузере
5. Следуйте инструкциям установщика

### Шаг 3: Установка темы

#### Через FTP (быстрее)

1. Подключитесь к хостингу через FTP (FileZilla)
2. Перейдите в `/wp-content/themes/`
3. Загрузите папку `wordpress-theme`
4. Переименуйте её в `switchonai`
5. Войдите в админ-панель WordPress
6. Перейдите в **Внешний вид → Темы**
7. Активируйте тему "Switch On AI"

#### Через админ-панель

1. Создайте ZIP-архив папки `wordpress-theme`
2. Войдите в админ-панель WordPress
3. Перейдите в **Внешний вид → Темы → Добавить новую**
4. Нажмите **Загрузить тему**
5. Выберите ZIP-файл
6. Нажмите **Установить**
7. Активируйте тему

### Шаг 4: Настройка темы

1. Перейдите в **Внешний вид → Настроить**
2. Настройте разделы:
   - **Идентификация сайта**: название и описание
   - **Hero Section**: главный экран
   - **Contact Information**: контакты
   - **Social Media**: ссылки на соцсети
3. Нажмите **Опубликовать**

### Шаг 5: Создание меню

1. Перейдите в **Внешний вид → Меню**
2. Создайте новое меню "Main Menu"
3. Добавьте произвольные ссылки:
   - Услуги → `#services`
   - Кейсы → `#cases`
   - Обучение → `#training`
   - FAQ → `#faq`
   - Контакты → `#contacts`
4. Назначьте в локацию "Primary Menu"
5. Сохраните меню

### Шаг 6: Настройка виджетов футера

1. Перейдите в **Внешний вид → Виджеты**
2. Добавьте виджет "Произвольный HTML" или "Меню" в каждую колонку
3. Настройте содержимое колонок (см. README темы)

### Шаг 7: Создание главной страницы

1. Перейдите в **Страницы → Добавить новую**
2. Название: "Главная"
3. Опубликуйте (содержимое не обязательно)
4. Перейдите в **Настройки → Чтение**
5. Выберите "Статическая страница"
6. Главная страница: "Главная"
7. Сохраните изменения

## 🔧 Редактирование контента

### Изменение текста на главной странице

Текст находится в файлах `template-parts/section-*.php`.

**Например, изменить услуги:**

1. Откройте файл `wordpress-theme/template-parts/section-services.php`
2. Найдите секцию с нужной услугой
3. Измените текст в HTML
4. Сохраните файл
5. Загрузите на хостинг (если используете FTP)
6. Обновите страницу сайта

### Изменение цветов

1. Откройте `wordpress-theme/style.css`
2. Найдите раздел `:root { ... }` в начале файла
3. Измените CSS-переменные:
   ```css
   --accent-blue: #69ace4;  /* Ваш цвет */
   --bg-primary: #000000;   /* Цвет фона */
   ```
4. Сохраните и загрузите файл

## 🐛 Частые проблемы

### Тема не отображается

**Решение:**
- Проверьте, что папка темы находится в `/wp-content/themes/`
- Убедитесь, что в папке есть файлы `style.css` и `index.php`
- Активируйте тему в **Внешний вид → Темы**

### Анимации не работают

**Решение:**
- Очистите кеш браузера (Ctrl+Shift+Delete)
- Проверьте консоль браузера на ошибки (F12)
- Убедитесь, что файл `assets/js/main.js` загружен

### Меню не появляется

**Решение:**
- Создайте меню в **Внешний вид → Меню**
- Назначьте его в локацию "Primary Menu"
- Добавьте пункты меню

### Главная страница показывает блог

**Решение:**
- Перейдите в **Настройки → Чтение**
- Выберите "Статическая страница"
- Назначьте нужную страницу как главную
