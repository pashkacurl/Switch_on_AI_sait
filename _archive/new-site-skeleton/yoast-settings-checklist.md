# Yoast SEO — настройки после установки
# switchonai.ru | 2026-03-29

Пройти последовательно сверху вниз.

---

## Шаг 1: Общее → Панель знаний

- Тип представленного сайта: **Физическое лицо**
- Имя: **Павел Лещенко**
- Логотип: загрузить фото (400×400 px минимум)

---

## Шаг 2: Общее → Возможности сайта

- Использовать Yoast SEO для: включить всё
- XML Sitemap: **ВКЛ**
- Breadcrumbs: **ВКЛ**

---

## Шаг 3: Типы контента

### Записи (Статьи блога)
- Показывать в результатах поиска: **ДА**
- Schema: Article → **BlogPosting**
- SEO title паттерн: `%%title%% — Включи ИИ`
- Meta description: оставить пустым (заполнять вручную)

### Страницы
- Показывать в результатах поиска: **ДА**
- SEO title паттерн: `%%title%% | Включи ИИ`

### Медиа/Вложения
- Показывать в результатах поиска: **НЕТ** (noindex)
- Редирект вложений → родительский пост: **ВКЛ**

---

## Шаг 4: Таксономии

### Категории
- Показывать в результатах поиска: **ДА**
- SEO title паттерн: `%%term_title%%: статьи об AI — Включи ИИ`
- Meta description паттерн: заполнить вручную для каждой

### Теги
- Показывать в результатах поиска: **НЕТ** (noindex)
- Причина: слишком много тегов = дубли

### Форматы
- Показывать в результатах поиска: **НЕТ**

---

## Шаг 5: Авторы

- Показывать страницы авторов: **НЕТ** (noindex)
- Или: разблокировать + заполнить профиль если нужна страница /o-avtore/

---

## Шаг 6: Архивы

- Архив по дате: **НЕТ** (noindex)
- Архив по автору: **НЕТ** (noindex)

---

## Шаг 7: XML Sitemap

Включить:
- [x] Записи
- [x] Страницы
- [x] Категории

Отключить:
- [ ] ~~Авторы~~ — ВЫКЛ
- [ ] ~~Теги~~ — ВЫКЛ
- [ ] ~~Форматы~~ — ВЫКЛ
- [ ] ~~Медиа~~ — ВЫКЛ

---

## Шаг 8: Breadcrumbs

- Разделитель: `›`
- Якорь главной страницы: `Главная`
- Показывать последний элемент как ссылку: **НЕТ**
- Включить schema breadcrumbs: **ДА**

---

## Шаг 9: Social / Open Graph

- Facebook OG: **ВКЛ**
- Twitter Card: **ВКЛ**
- Twitter Card тип: Summary with large image
- Изображение по умолчанию: загрузить og-default.jpg (1200×630)

---

## Шаг 10: Robots.txt и .htaccess

- Редактировать robots.txt через Yoast: **использовать готовый файл из skeleton**
- НЕ давать Yoast перезаписывать robots.txt автоматически

## Финальная проверка после настройки

1. Открыть: https://switchonai.ru/sitemap_index.xml — убедиться что работает
2. Открыть: https://switchonai.ru/robots.txt — убедиться что загружен правильный
3. Проверить schema: https://search.google.com/test/rich-results
4. Проверить security headers: https://securityheaders.com/?q=switchonai.ru
5. Отправить sitemap в Google Search Console
6. Отправить sitemap в Яндекс Вебмастер
