## Лучшие практики дизайна

### Цветовые схемы

**Тёмная тема (наиболее популярна):**
```
Фон: #000000 - #0a0c1c
Текст: #ffffff
Акцент 1: синий/голубой (#5568f5, #69ace4)
Акцент 2: оранжевый/жёлтый (#ffd669, #FE7524)
```

**Светлая тема:**
```
Фон: #ffffff
Текст: #111111
Акцент: синий (#016FDC) или красный
```

### Типографика
- **Заголовки:** Montserrat, Inter (600-800 weight)
- **Основной текст:** Roboto, PT Sans, Inter (400-500 weight)
- **Размеры:** H1: 48-80px, H2: 30-40px, Body: 16-18px

### UI-элементы

**Кнопки:**
- Border-radius: 30-60px (сильно скруглённые)
- Градиентные или однотонные
- Hover-эффекты с изменением цвета
- Высота: 46-64px

**Карточки:**
- Border-radius: 20-30px
- Box-shadow с цветным свечением
- Полупрозрачные фоны (backdrop-filter: blur)

### Анимации и эффекты
- Blur-фильтры на фоновых элементах
- Плавные hover-переходы (0.2-0.3s)
- Градиентные переливы
- Вращающиеся границы (для премиальных элементов)

### Структура Hero-секции
1. Крупный заголовок (проблема + решение)
2. Подзаголовок с конкретикой
3. CTA-кнопка (яркий акцент)
4. Визуал продукта или абстрактный фон
5. Социальное доказательство (логотипы клиентов)

### Обязательные секции
1. Hero с УТП
2. Проблема/Решение
3. Услуги/Продукты (карточки)
4. Как это работает (процесс)
5. Кейсы/Результаты
6. Отзывы
7. Тарифы
8. FAQ
9. CTA-блок
10. Футер с контактами

## Рекомендации для идеального дизайна

### Выбор стиля
Рекомендую **тёмную тему** — она:
- Ассоциируется с технологичностью и AI
- Позволяет использовать яркие акценты
- Выглядит премиально
- Используется большинством лидеров рынка

### Цветовая палитра (рекомендация)
```
Основной фон: #0a0c1c (тёмно-синий)
Вторичный фон: #1a1a2e
Текст: #ffffff
Акцент 1: #5568f5 (синий) — кнопки, ссылки
Акцент 2: #ffd669 (жёлтый) — выделения, badges
Акцент 3: #18E896 (зелёный) — успех, статусы
```

### Шрифты (рекомендация)
```
Заголовки: Montserrat или Inter (700 weight)
Текст: Inter или Roboto (400-500 weight)
```

### Ключевые элементы
1. **Фиксированный хедер** с прозрачным фоном
2. **Анимированный Hero** с градиентами
3. **Карточки услуг** со свечением
4. **Видео-демо** продукта
5. **Калькулятор/Конфигуратор** услуг
6. **Live-чат/бот** для демонстрации
7. **Интеграции** (логотипы Telegram, WhatsApp и т.д.)

### Что делает сайт выделяющимся
- Интерактивные демо продукта прямо на сайте
- Конкретные цифры результатов (не общие фразы)
- Видео-кейсы клиентов
- Уникальные анимации (не стандартные)
- Быстрая загрузка

## Итоговые рекомендации по дизайну

### Цветовая палитра
```css
--bg-primary: #0a0c1c;      /* Основной тёмный фон */
--bg-secondary: #1a1a2e;    /* Вторичный фон для карточек */
--bg-card: #0f1129;         /* Фон карточек */
--text-primary: #ffffff;    /* Основной текст */
--text-secondary: #9a9a9a;  /* Вторичный текст */
--accent-blue: #5568f5;     /* Кнопки, ссылки */
--accent-yellow: #ffd669;   /* Выделения, badges */
--accent-green: #18E896;    /* Успех, статусы */
--gradient: linear-gradient(135deg, #5568f5, #a234fd);
```

### Типографика
```css
/* Заголовки */
font-family: 'Montserrat', sans-serif;
font-weight: 700;
h1: 56-72px;
h2: 36-48px;
h3: 24-30px;

/* Основной текст */
font-family: 'Inter', sans-serif;
font-weight: 400-500;
body: 16-18px;
```

### UI-компоненты

**Кнопки:**
- Primary: градиент синий-фиолетовый, border-radius: 40px
- Secondary: прозрачный фон, border: 2px solid #5568f5
- Hover: свечение и небольшое увеличение

**Карточки услуг:**
- Фон: rgba(15, 17, 41, 0.8)
- Border-radius: 24px
- Border: 1px solid rgba(85, 104, 245, 0.2)
- Hover: свечение синим

**Секции:**
- Чередование тёмного фона с градиентными акцентами
- Blur-эффекты на декоративных элементах
- Полноширинные блоки
