> Дата создания: 23 января 2026
> Проект: Switch On AI / Включи ИИ

## Почему это важно

- 3D-элементы создают **ощущение глубины и премиальности**
- Scroll-анимации **удерживают внимание** и направляют пользователя
- Интерактивность **выделяет сайт** среди конкурентов
- AI-тематика отлично сочетается с футуристичными 3D-эффектами

**Статистика:**
- 94% первых впечатлений основаны на дизайне
- 88% пользователей не вернутся после плохого опыта
- Сайты с анимациями имеют на 30% выше конверсию

## Типы анимаций для сайта

### 1. Scroll-triggered анимации
Элементы появляются/анимируются при прокрутке страницы.

**Эффекты:**
| Эффект | Описание | Где использовать |
|--------|----------|------------------|
| **Fade-in** | Плавное появление снизу/сбоку | Карточки, текст |
| **Scale** | Увеличение при входе в viewport | Изображения, иконки |
| **Parallax** | Слои двигаются с разной скоростью | Фоны, декор |
| **Reveal** | "Раскрытие" по маске | Заголовки, фото |
| **Pin** | Секция "прилипает" к экрану | Процессы, таймлайны |
| **Stagger** | Последовательное появление | Списки, карточки |

### 2. 3D-объекты
Интерактивные 3D-модели на странице.

**Идеи для AI-сайта:**
- **3D робот/ассистент** — вращается при скролле или движении мыши
- **3D мозг/нейросеть** — визуализация AI с частицами и связями
- **3D планета/сфера** — символ глобальности и технологий
- **3D абстрактные формы** — морфинг геометрии
- **3D логотип** — объёмная версия лого "Включи ИИ"

### 3. Hero-секция с 3D

**Вариант A: Фоновая 3D-сцена с частицами**
```
- Плавающие частицы/точки, соединённые линиями
- Реагируют на движение мыши
- Создают эффект "нейронной сети"
- Производительно, работает на мобильных
```

**Вариант B: 3D-объект по центру**
```
- Вращающаяся геометрия или робот
- При скролле объект отъезжает/растворяется
- Появляется контент страницы
- Впечатляюще, но требует оптимизации
```

**Вариант C: Интерактивная сфера**
```
- Сфера с текстурой данных/кода
- При движении мыши — вращение
- При скролле — zoom-in внутрь сферы
- Эффект погружения в мир AI
```

### 4. Морфинг и трансформации
**Применение:**
- Логотип трансформируется при загрузке
- Иконки услуг "оживают" при наведении
- Плавные переходы между секциями
- Числа "счётчики" (animating numbers)

### Для scroll-анимаций

#### GSAP + ScrollTrigger (ЛУЧШИЙ ВЫБОР)

**Почему GSAP:**
- Индустриальный стандарт
- Плавные анимации 60fps
- Огромное сообщество
- Отличная документация

**Базовый пример:**
```javascript
// Элемент появляется при скролле
gsap.from(".card", {
  scrollTrigger: {
    trigger: ".card",
    start: "top 80%",
    end: "bottom 20%",
    scrub: true
  },
  opacity: 0,
  y: 100,
  duration: 1
});

// Stagger — карточки появляются по очереди
gsap.from(".service-card", {
  scrollTrigger: {
    trigger: ".services-section",
    start: "top 70%"
  },
  opacity: 0,
  y: 50,
  stagger: 0.2,
  duration: 0.8
});

// Pin — секция прилипает
gsap.to(".horizontal-section", {
  scrollTrigger: {
    trigger: ".horizontal-section",
    start: "top top",
    end: "+=3000",
    scrub: 1,
    pin: true
  },
  x: "-200vw"
});
```

#### Сравнение библиотек

| Библиотека | Плюсы | Минусы | Для кого |
|------------|-------|--------|----------|
| **GSAP** | Мощная, плавная, много функций | Нужен JS | Профи |
| **AOS** | Простая, легковесная | Ограниченные эффекты | Новички |
| **Lenis** | Сверхплавный скролл | Дополнение к GSAP | Профи |
| **ScrollReveal** | Простая настройка | Базовые эффекты | Новички |
| **Locomotive Scroll** | Smooth scroll + parallax | Тяжеловата | Средний |

## Пошаговая реализация 3D в WordPress

### Шаг 1: Создание 3D-объекта в Spline

1. Зарегистрироваться на https://spline.design
2. Создать новый проект
3. Настроить сцену:
   - Background: #0a0c1c (под цвет сайта)
   - Добавить объект (Sphere, Torus, или кастомный)
4. Настроить материалы:
   - Glass — для прозрачности
   - Gradient — для переливов
   - Matcap — для металлического эффекта
5. Добавить освещение:
   - Directional Light
   - Ambient Light (мягкий)
6. Настроить взаимодействия:
   - Events → Mouse Hover → Rotate
   - Events → Scroll → Scale/Move

### Шаг 2: Экспорт из Spline

1. File → Export → Viewer (Public URL)
2. Или: File → Export → Code (для iframe)
3. Скопировать ссылку/код
4. Опционально: настроить loading="lazy"

### Шаг 3: Интеграция в WordPress

**Вариант A — Через плагин Nexter Blocks:**
1. Установить Nexter Blocks
2. В Gutenberg добавить блок "Spline Viewer"
3. Вставить Public URL
4. Настроить размеры

**Вариант B — Через Elementor HTML:**
```html
<div class="spline-container">
  <spline-viewer
    url="https://prod.spline.design/YOUR-ID/scene.splinecode"
    loading="lazy">
  </spline-viewer>
</div>

<style>
.spline-container {
  width: 100%;
  height: 600px;
}
spline-viewer {
  width: 100%;
  height: 100%;
}
</style>
```

### Шаг 4: Добавление scroll-анимаций

**Через плагин Motion.page (визуально):**
1. Установить Motion.page ($69)
2. Открыть любую страницу
3. Кликнуть на элемент → Add Animation
4. Выбрать тип (Fade, Scale, Move)
5. Настроить триггер (Scroll, Load, Hover)
6. Сохранить

**Через кастомный JS (GSAP):**

Добавить в functions.php:
```php
function enqueue_gsap_scripts() {
    // GSAP Core
    wp_enqueue_script(
        'gsap',
        'https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/gsap.min.js',
        array(),
        '3.12.2',
        true
    );

    // ScrollTrigger Plugin
    wp_enqueue_script(
        'scroll-trigger',
        'https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/ScrollTrigger.min.js',
        array('gsap'),
        '3.12.2',
        true
    );

    // Кастомные анимации
    wp_enqueue_script(
        'custom-animations',
        get_stylesheet_directory_uri() . '/assets/js/animations.js',
        array('gsap', 'scroll-trigger'),
        '1.0',
        true
    );
}
add_action('wp_enqueue_scripts', 'enqueue_gsap_scripts');
```

Создать файл `/assets/js/animations.js`:
```javascript
// Регистрация ScrollTrigger
gsap.registerPlugin(ScrollTrigger);

// Fade-in для всех элементов с классом .fade-in
gsap.utils.toArray('.fade-in').forEach(elem => {
  gsap.from(elem, {
    scrollTrigger: {
      trigger: elem,
      start: "top 85%",
      toggleActions: "play none none none"
    },
    opacity: 0,
    y: 50,
    duration: 0.8,
    ease: "power2.out"
  });
});

// Stagger для карточек услуг
gsap.from('.service-card', {
  scrollTrigger: {
    trigger: '.services-section',
    start: "top 70%"
  },
  opacity: 0,
  y: 60,
  stagger: 0.15,
  duration: 0.6,
  ease: "power2.out"
});

// Параллакс для фоновых элементов
gsap.utils.toArray('.parallax-bg').forEach(bg => {
  gsap.to(bg, {
    scrollTrigger: {
      trigger: bg,
      start: "top bottom",
      end: "bottom top",
      scrub: 1
    },
    y: -100,
    ease: "none"
  });
});

// Timeline для Hero-секции
const heroTl = gsap.timeline();
heroTl
  .from('.hero-title', { opacity: 0, y: 30, duration: 0.8 })
  .from('.hero-subtitle', { opacity: 0, y: 20, duration: 0.6 }, "-=0.4")
  .from('.hero-cta', { opacity: 0, scale: 0.9, duration: 0.5 }, "-=0.3");
```

## Оптимизация производительности

### Проблемы и решения

| Проблема | Решение |
|----------|---------|
| 3D тормозит на мобильных | Mobile fallback — статичное изображение |
| Долгая загрузка | Lazy loading + preload для hero |
| Анимации дёргаются | will-change: transform; + GPU acceleration |
| Высокий CPU | Reduce motion для пользователей с настройкой |
| Большой размер моделей | Сжатие в Spline + GLB optimization |

### Рекомендуемые плагины кэширования
- **LiteSpeed Cache** (бесплатно, если LiteSpeed сервер)
- **WP Rocket** ($59/год)
- **Perfmatters** (отключение ненужных скриптов)

### CSS для оптимизации
```css
/* GPU acceleration */
.animated-element {
  will-change: transform, opacity;
```
